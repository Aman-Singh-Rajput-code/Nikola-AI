# Nikola AI

## Vision

Create an AI Operating System similar to Jarvis.

Nikola is NOT a chatbot.

Nikola is an extensible AI platform capable of reasoning,
planning, remembering, and interacting with the operating
system through modular tools.

---

- Provider Independent
- Modular
- Testable
- Production Ready
- Clean Architecture
- Enterprise Quality
